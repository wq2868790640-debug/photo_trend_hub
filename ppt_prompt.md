# PPT 思路梳理请求

你是一位资深产品经理兼视觉设计师。请帮我梳理一个 5 分钟演讲 PPT 的内容框架，演讲主题是"如何用 AI Agent 从 0 搭建一个专业摄影趋势中台，花了 0 行代码"。

---

## 我的项目背景

### 这是什么

Photo Trend Hub — 一个「当代摄影与商业摄影趋势中台」网站。它是一个私人档案工具，用于：

- 采集我浏览到的摄影网站、摄影师主页、时尚大片、策展平台
- 用专业维度标注（布光流派、调色基因、情绪权重、商业降维周期）
- 交叉过滤检索
- 一键导出 Markdown 趋势周报

### 为什么做（三个真实痛点）

1. **灵感散落一地**：看到好作品 → 收藏网页、截屏、记笔记 → 散在书签栏/相册/备忘录，下次找不到
2. **缺乏专业归类**：收藏了一堆，没按"商业片还是艺术片""用了什么光""什么调色风格"分类，想不起来这条信息是干嘛的
3. **趋势判断靠直觉**："这种风格明年会不会火？"——全靠拍脑袋，品牌问你为什么选这个参考，你说不上来

### 怎么做的（全程 AI Agent 搭建）

我一行代码都没写。全程用自然语言跟 AI Agent 对话：

- 我描述需求 → Agent 几秒钟生成代码 → 我打开看效果 → 提修改意见 → Agent 改
- 多轮迭代，从想法到可用产品约 2 小时
- 最终产物：1 个 HTML 文件（1,800+ 行），双击浏览器即用
- 部署：GitHub Pages，零成本，永久公网链接

### 技术细节

- 纯前端：HTML + Tailwind CSS + 原生 JavaScript
- 存储：IndexedDB（支持数百条带 Base64 图片的记录，告别 localStorage 5MB 瓶颈）
- 数据持久化：JSON 文件自动导入，支持导出备份
- 视觉风格：美术馆白墙 · 杂志 Editorial 范式（Cormorant Garamond 衬线 + Inter 无衬线）
